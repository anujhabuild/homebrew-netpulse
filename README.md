# homebrew-netpulse

Homebrew tap for [NetPulse](https://github.com/anujhabuild/netpulse), a macOS menu bar network bandwidth monitor.

> ⚠️ **Do not delete this repository.** This is the Homebrew tap — it's exactly what `brew tap anujhabuild/netpulse` clones, and it's what every Homebrew user's `brew install`/`brew upgrade --cask netpulse` reads to find the app. Deleting it breaks Homebrew installs and upgrades for everyone, even though the app's actual source and releases (in [`netpulse`](https://github.com/anujhabuild/netpulse)) would be untouched.

## Install

```
brew tap anujhabuild/netpulse
brew install --cask netpulse
```
